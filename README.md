# Atlasora

A modern travel discovery platform built with React, JavaScript, Tailwind CSS, and Vite. The platform allows users to explore countries, discover famous attractions, read travel guides, search destinations, and save favorite places using real API data.

## Live Demo

https://atlasoratravelexplorer.netlify.app/

## Features

- Explore countries from around the world
- Search countries, cities, landmarks, and attractions
- Country explorer with flags, capitals, currencies, languages, and population data
- Featured destinations section
- Trending destinations section
- Famous attractions and landmarks showcase
- Curated travel collections and travel inspiration
- Travel guides and destination information
- Save favorite destinations using Local Storage
- Dark mode support
- Responsive design for desktop, tablet, and mobile devices
- Dynamic API data integration
- Modern and immersive user interface

## Technologies Used

- React.js
- JavaScript (ES6+)
- Tailwind CSS
- Vite
- HTML5
- REST Countries API
- Wikipedia API
- Local Storage

## Project Structure

- Component-based React architecture
- API integration and data fetching
- State management using React Hooks
- Search and filtering functionality
- Favorites management with Local Storage
- Responsive design implementation
- Modular and reusable component organization

## Project Goal

This project was developed to strengthen frontend development skills by building a realistic travel discovery platform using React and real-world APIs. It focuses on API integration, state management, search functionality, responsive design, component reusability, and creating modern user experiences.
